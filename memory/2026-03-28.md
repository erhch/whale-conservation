```
2d389a9 feat(common): 添加 CacheInterceptor 响应缓存拦截器
```

**技术细节:**

```typescript
// 基础用法 - 默认 5 分钟缓存
@UseInterceptors(CacheInterceptor)
@Get('species')
findAllSpecies() {
  return this.speciesService.findAll();
}

// 自定义缓存时间 - 1 小时
@UseInterceptors(CacheInterceptor)
@CacheTTL(3600)
@Get('statistics')
getStatistics() {
  return this.statsService.getStatistics();
}

// 自定义缓存键
@UseInterceptors(CacheInterceptor)
@CacheKey('home-page-data')
@Get('home')
getHomeData() {
  return this.homeService.getData();
}
```

**适用场景:**
- 物种列表查询 (数据变化频率低)
- 统计分析接口 (计算开销大)
- 监测站点数据 (读多写少)
- 首页聚合数据 (多表查询)

**注意事项:**
- 数据更新后需手动清除相关缓存
- 敏感数据不建议缓存
- 生产环境建议切换为 Redis 缓存

**技术细节:**

```typescript
// 基础用法 - 无限制
@Query('weight', ParseOptionalFloatPipe)
weight?: number;

// 带默认值
@Query('minLength', new ParseOptionalFloatPipe({ defaultValue: 0 }))
minLength: number;

// 带范围验证 - 水温 0-40°C
@Query('waterTemp', new ParseOptionalFloatPipe({ min: 0, max: 40 }))
waterTemp?: number;

// 带精度限制 - 盐度最多 2 位小数
@Query('salinity', new ParseOptionalFloatPipe({ precision: 2 }))
salinity?: number;

// 组合使用 - 鲸鱼体长 1-35 米，最多 2 位小数
@Query('length', new ParseOptionalFloatPipe({ min: 1, max: 35, precision: 2 }))
length?: number;
```

**适用场景:**
- 鲸鱼体长/体重查询 (范围验证)
- 水温/盐度/PH 值筛选 (环境参数)
- 地理坐标精度控制 (经纬度)
- 统计数据的容差范围

**与 ParseOptionalIntPipe 的对比:**
| 特性 | IntPipe | FloatPipe |
|------|---------|-----------|
| 数据类型 | 整数 | 浮点数 |
| 精度验证 | ❌ | ✅ (precision) |
| 典型场景 | 分页、数量 | 测量数据、坐标 |
